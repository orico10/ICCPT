# User Execution (Terminal)

This section explains how to run ICCPT as a user from the terminal.

## Windows 
The packaged executable is created for Windows.

1. Open Command Prompt.
2. Go to the project folder (the same level as `main.py`).
3. Run:

```cmd
main.exe
```

### Important note

The Windows executable should always remain at the same level as `main.py`
inside the `CleanCooking_os` project folder.

## macOS

The repository does not ship a macOS executable by default.
If you need one, see the developer section on building executables for macOS.
