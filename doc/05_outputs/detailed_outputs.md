# Detailed Outputs

Detailed outputs are the primary analysis tables stored in `data/outputs`.

They are intended for downstream analysis, visualization, and dashboards.

## Current exports

From the existing export flow, the repository includes examples such as:

- `data/outputs/potential_adop_for_visualization.tsv`
- `data/outputs/merged_opex_capex_yearly.tsv`

Additional export helpers exist for:

- electrified area cost exports
- income-model exports
- electricity-cost exports
- country electricity demand summaries

Some of these helpers may be optional or not enabled in the current export path.

## When to use them

- Build dashboards in tools like Power BI or Tableau.
- Map adoption or demand by area and technology.
- Analyze cost structures and financial progression over time.
