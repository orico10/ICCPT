# Summary Outputs

Summary outputs are quick validation tables stored in `data/summary_results`.

They are intended to help users validate a run before doing deeper analysis.

## Examples

The repository already includes examples such as:

- `SocialImpact.tsv`
- `CountryAdoptionPotentialByTech.tsv`
- `CostOfCooking.tsv`

## When to use them

- Verify that a scenario ran correctly.
- Compare high-level outcomes across scenarios.
- Feed quick checks into the Excel interface before running a full analysis.
